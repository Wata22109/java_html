    はじめに

    JavaScriptは動的型付け言語で、ウェブサイトに対話性を追加するプログラミング言語です。
    このページでは、JavaScriptの基本的な構文について、プログラムの最低限の知識を知っていることを前提に解説します。

    
// 変数宣言
let greeting = "Hello, World!";

// 関数定義
function greet(name) {
    console.log(`${greeting} My name is ${name}.`);
}

// 関数呼び出し
greet("Alice");

// オブジェクト操作
let person = {
    name: "Bob",
    age: 30,
    sayHello: function() {
        console.log(`Hello, I'm ${this.name}`);
    }
};

person.sayHello();

    変数、定数、配列

    変数宣言にはlet、定数宣言にはconstを使います。プログラム中に変更する予定がある場合は変数、 変更させたくない場合は定数を使用します。定数を再代入しようとするとエラーが発生します。

    
// 変数宣言
let x = 10;
let y = 20;

x = 30; // 再代入可能

// 定数宣言
const PI = 3.14;
const TAX_RATE = 0.08;

PI = 3.14159; // エラー(定数は誤って再代入することを防ぐ)

    Node.jsの概要と活用方法

    Node.jsはサーバーサイドJavaScriptの実行環境で、非同期I/Oと単一スレッドのイベントループが特徴です。
